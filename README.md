# 🐾 VET24 – Veterinary Web Map Application

A web-based GIS application for locating 24-hour veterinary clinics, developed using Leaflet.js and OpenStreetMap.

---

## 🚀 Project Overview

VET24 is a **Web Map system** that integrates geospatial data with a web interface to help users find nearby veterinary clinics.  
The application visualizes location-based data (GPS coordinates) on an interactive map and provides detailed clinic information.

---

## Web Map Concept (System Understanding)

This project follows a typical **Web Map Architecture**:

### Client Side (Frontend)
- Displays map using Leaflet.js  
- Renders markers and user interface  
- Handles user interaction (click, popup)  

### Map Service
- Uses OpenStreetMap as base map (tile layer)  

### Data Layer
- GeoJSON stores spatial data (latitude, longitude)  
- Contains clinic information (name, address, services)  

### Server (Local via XAMPP)
- Serves HTML/CSS/JS files  
- Connects frontend with data  

📌 This demonstrates understanding of how **GIS + Web Development** work together in real-world systems.
